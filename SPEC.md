# Android RPA 软件规格说明书

## 1. 项目概述

**项目名称**: AutoBot RPA  
**项目类型**: Android 原生应用  
**核心功能**: 类似按键精灵的RPA自动化工具，提供按键、触摸、截图、图像识别和流程控制功能，用户可通过组合操作实现游戏和应用的自动化。

## 2. 技术栈选择

### 框架与语言
- **语言**: Kotlin 1.9.x
- **最低SDK**: Android API 26 (Android 8.0)
- **目标SDK**: Android API 34 (Android 14)
- **编译SDK**: Android API 34

### 主要框架/库
- **UI**: Jetpack Compose + Material Design 3
- **依赖注入**: Hilt
- **异步**: Kotlin Coroutines + Flow
- **持久化**: Room Database
- **图像识别**: OpenCV Android SDK / Template Matching
- **访问性服务**: Android AccessibilityService

### 状态管理
- ViewModel + StateFlow

## 3. 功能列表

### 3.1 核心动作操作
- **触摸操作**: 点击、滑动、长按、双击
- **按键操作**: 模拟物理按键(返回、主页、菜单等)
- **输入操作**: 文本输入
- **延时操作**: 等待指定时间

### 3.2 图像识别
- **截图**: 截取当前屏幕
- **图像匹配**: 模板匹配查找图像位置
- **颜色识别**: 获取指定坐标颜色

### 3.3 流程控制
- **循环**: 重复执行N次或无限循环
- **条件判断**: 图像是否存在、颜色判断
- **跳转**: 标签和跳转指令
- **子流程**: 脚本调用子脚本

### 3.4 脚本管理
- **创建/编辑/删除**: 脚本的完整生命周期管理
- **导入/导出**: 脚本的备份和恢复
- **分组**: 按功能分类管理脚本

### 3.5 脚本执行
- **运行/暂停/停止**: 运行时控制
- **日志输出**: 执行过程日志
- **执行统计**: 运行次数、成功/失败统计

## 4. UI/UX 设计方向

### 整体风格
- Material Design 3 设计语言
- 简洁现代的界面
- 深色模式支持

### 颜色方案
- 主色调: 蓝色系 (#1976D2)
- 辅助色: 绿色表示成功 (#4CAF50)
- 警告色: 橙色 (#FF9800)
- 错误色: 红色 (#F44336)

### 布局方案
- **底部导航**: 脚本列表、执行控制、设置
- **脚本编辑**: 列表+详情模式
- **动作块**: 可拖拽的卡片式设计
- **浮动操作**: 快速执行按钮

### 关键界面
1. **主界面**: 脚本列表 + 快速执行悬浮按钮
2. **脚本编辑**: 动作块列表 + 添加动作面板
3. **执行界面**: 日志输出 + 控制按钮
4. **设置界面**: 通用设置 + 权限管理
